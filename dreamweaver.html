What I learned on Web Modules

HTML CSS Adobe Dreamweaver
HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications. CSS stands for Cascading Style Sheets. It is a style sheet language used for describing the presentation of a document written in HTML. Adobe Dreamweaver is a proprietary web development tool developed by Adobe Inc. It provides a visual interface for building websites and web applications.
HTML provides the structure of the webpage. CSS is used to style the HTML elements. Dreamweaver offers features such as code highlighting, syntax checking, and visual design tools.
HTML elements are represented by tags. CSS allows for the separation of content from presentation. It supports various programming languages and frameworks.
Learning HTML I understood basic tags such as <html>, <head>, <body>, and learn about semantic elements like <header>, <nav>, <section>, etc. Mastering CSS, I learned how to style individual and groups of elements using various visual CSS rules. For Adobe Dreamweaver, I became familiar with its interface, features like Live View, Code View.
I learnt about HTML5 features such as new structural elements (<header>, <footer>, <article>, etc.), forms, multimedia. I learned how to add styles websites with CSS and how to use selectors to apply styles to specific elements. Practice building websites and applications using Dreamweaver, and understand best practices for organizing files and assets.
Understood the importance of accessibility in HTML, and learned to optimize my code for better performance. Experimented with CSS rules for displaying and positioning elements on my web page. Stayed updated with the latest web design trends and technologies, and continuously improved my skills in web development.
